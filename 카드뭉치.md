# 카드뭉치 

<div>두가지 리스트를 한번에 탐색하는 방법을 구현해낼 수 있는지 묻는 문제이다.</div>

# 첫번째 시도(50/100)

~~~python
def solution(cards1, cards2, goal):
    
    cnt = 0
    
    for k in goal:
        if cards1[0] == k:
            cnt += 1
            cards1 = cards1[1:]
        elif cards2[0] == k:
            cnt += 1
            cards2 = cards2[1:]
        else:
            return "No"
    
    if cnt == len(goal):
        return "Yes"
~~~

<div>List index out of range에러가 발생하였다. 리스트의 인덱스로 접근하는 경우 해당 에러에 주의할 것.</div>
